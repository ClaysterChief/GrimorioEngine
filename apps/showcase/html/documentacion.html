◆ Introducción

Grimorio Engine es un framework CSS puro. Todo el estilo vive en un único archivo (grimorio.css) y las interacciones opcionales en grimorio.js. No necesitas build para desarrollar.

◆

Regla central: cero style="". Todo se resuelve con utilidades o clases BEM.

i

Nota: las fuentes se importan dentro del propio CSS vía Google Fonts.

◆ Instalación

Instala desde GitHub (sin publicar en npm):

bash
npm install github:ClaysterChief/GrimorioEngine

Luego enlaza el CSS y el JS opcional:

html
<link rel="stylesheet" href="css/grimorio.css" />
<script defer src="js/grimorio.js"></script>
