COMPONENTE · Acordeón

Multi-panel — varios abiertos

Todos los colores del framework referencian custom properties (--acento, --void-surface, etc.) para adaptarse automáticamente al cambiar la paleta activa.
--font-display (Orbitron 900) para títulos · --font-body (VT323) para cuerpo · --font-mono (Share Tech Mono) para código y etiquetas.
Las esquinas afiladas son intencionales a la estética PS1. No usar border-radius en elementos estructurales del framework.

Exclusivo — estilo FAQ

Framework CSS puro con estética PS1/CRT. Sin dependencias JavaScript externas. Una sola hoja de estilos: packages/css/grimorio.css.
Añade la clase .cosmos o .crimson al <body> o cualquier contenedor. Sin clase = paleta Dual Signal por defecto.
No. Abre index.html directamente en el navegador. El script npm run build solo genera el CSS minificado.
BEM simplificado, todo en español, kebab-case. Prefijos principales: btn, campo-, tarjeta-, alerta-, modal-.
